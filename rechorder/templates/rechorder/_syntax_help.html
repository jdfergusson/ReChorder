Verses/Sections


Sections are named using curly brackets:
{<Type><Number><!>}
Each section should have a unique name

Type


The type is what is expanded into the name of the section:
  • v→Verse
  • c→Chorus
  • b→Bridge
  • p→Prechorus
  • i→Intro
  • o→Ending
  • m→Instrumental

Number


The number is the section number, for example Chorus 1 should be {c1}, verse 3 should be {v3}

Exclamation mark


The exclamation mark at the end of the section name tells the system that this section should not be included when exporting lyrics to OpenLP. For example an introduction that contains no lyrics, followed by a verse, might look like this:
{i!}
[G] [D] [Em] [C]

{v1}
[G]Start of a [D]verse...

Chords


Chords go in square brackets inline with the lyrics
[<Note><Modifier>/<Bass Note>].
Everything apart from the Note is optional. Notes should be capitalised

Some examples:

[G]Some [Fm]really meaningful [Bmb5/D]Lyr[E/F#]ics

Verse Order


The verse order is used only when exporting the song to OpenLP. It tells OpenLP in which order to display the verses. The labels used should match the section labels used in the song.

For example:

v1 c1 v2 c1 b c1 c2 o

Formatting


Optional Line Breaks


You can add optional line breaks using a backslash "\", for when some devices may want short lines, but others may want long. For instance, take this line:
All I once held dear, built my life upon
Some people may want to view this split across two lines:
All I once held dear,
built my life upon
To allow for this, use the backslash when writing the song to enter a line break that may or may not be displayed. E.G.:
[C]All [G/B]I [Am]once [F]held [C]dear, \built my [F]life [G]up[C]on,
All [G/B]this [Am]world [F]re[C]veres \and [Am]wars to [G]own;
The settings page allows you to change whether the optional line breaks are used or not. The optional line breaks are always used in the OpenLP export of a song.

Optional sub-section splits


To indicate a sub section (for instance a phrase break within a verse) you can use an empty line. This will be used by OpenLP to split pages more sensibly. For example:
[E][F#m7][B] [A]Heal my heart and make it [B]clean
[E]Open up my eyes to the [C#m]things unseen
[A]Show me how to love like [B]You have loved me [C#m]

[A]Break my heart for what breaks [B]Yours
[E]Everything I am for Your [C#m]kingdom's cause
[A]As I walk from earth in[B]to eternity [C#m]
